# ApkgGenerator

Auto-generate note.apkg for [Anki](https://apps.ankiweb.net) by users' `json` file.

# Features

- [ ] Command line interface.
- [ ] Get dict from [ECDICT](https://github.com/skywind3000/ECDICT)
- [ ] AI support: GPT
- [ ] Build `.apkg` with preset template
